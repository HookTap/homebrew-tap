## Install

```bash
brew install HookTap/tap/hooktap
```

Or tap first, then install:

```bash
brew tap HookTap/tap
brew install hooktap
```

## Available tools

| Tool | Description | Repository |
|------|-------------|------------|
| [`hooktap`](https://github.com/HookTap/hooktap-cli) | Send webhook events to the HookTap app from your terminal | [HookTap/hooktap-cli](https://github.com/HookTap/hooktap-cli) |

## Updating

```bash
brew update
brew upgrade hooktap
```

## About

The casks in this tap are generated and published automatically by [GoReleaser](https://goreleaser.com) whenever a new version is released — please don't edit them by hand. To report an issue or request a change, open it on the tool's own repository.
